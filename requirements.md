# 올티칭 영어교습소 랜딩 페이지 제작 요구사항

## 1. 프로젝트 개요

### 1.1 프로젝트 목적
광주 광산구 신창동에 위치한 **올티칭 영어교습소**를 홍보하고, 학부모의 학생 등록 상담(전화·카카오톡)으로 이어지도록 유도하는 **원 페이지(One-Page) 랜딩 페이지** 제작.

### 1.2 핵심 메시지
> "매일의 작은 노력이 쌓여 큰 실력이 됩니다."
- 단기간 선행학습이 아닌, **꾸준한 학습 습관과 루틴 형성**을 통한 자기주도 학습 역량 강화
- **원장 직강 + 소수정예**의 프리미엄 밀착 관리

### 1.3 타겟 사용자
- 광주 광산구 신창동·수문동 일대의 초·중등 학부모 (주 30~50대)
- 근처 초·중학교(수문초, 신창중 등) 재학생 학부모

---

## 2. 기술 스택 및 개발 방향

### 2.1 개발 방식
- **Claude Code 기반 바이브 코딩(Vibe Coding)**
- 단일 페이지 정적 사이트, 스크롤 기반 섹션 구성

### 2.2 권장 기술 스택
- **프레임워크**: Next.js 14 (App Router) 또는 Vite + React
- **스타일링**: Tailwind CSS
- **애니메이션**: Framer Motion (스크롤 인터랙션), AOS 또는 Intersection Observer
- **아이콘**: lucide-react
- **폰트**: Pretendard (Google Fonts / CDN)
- **배포**: Vercel

### 2.3 반응형 요구사항
- **모바일 우선(Mobile-First)** 설계 (학부모 트래픽 대부분이 모바일)
- 브레이크포인트: `sm(640)`, `md(768)`, `lg(1024)`, `xl(1280)`
- 모바일에서 CTA 버튼(전화/카톡)이 **화면 하단 고정(sticky bottom bar)**

## 7. 접근성 (a11y)

- 모든 이미지에 의미 있는 `alt` 텍스트
- CTA 버튼은 `<a>` 또는 `<button>` 시멘틱 태그 사용
- 색 대비 WCAG AA 준수 (본문 텍스트 대비 4.5:1 이상)
- 키보드 탐색 가능 (`focus-visible` 스타일 명확히)
